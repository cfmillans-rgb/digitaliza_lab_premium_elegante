# Digitaliza Lab — Landing Premium

## Archivos
- `index.html`
- `assets/css/style.css`
- `assets/js/script.js`
- `assets/img/logo.png`

## Para probar
1. Descomprime el ZIP.
2. Abre `index.html` en tu navegador.
3. Cambia el número de WhatsApp en:
   - `assets/js/script.js`
   - `index.html` (botón flotante y footer)

## Ajustes rápidos
- Hero, textos y FAQ: `index.html`
- Colores y estilos: `assets/css/style.css`
- Formulario a WhatsApp y contadores: `assets/js/script.js`

## Notas
- Usa Bootstrap 5 y Bootstrap Icons vía CDN.
- El diseño está alineado a una estética premium oscura con acento naranjo, tomando como referencia la identidad visual de Digitaliza Lab.
- La promesa principal quedó en 72 horas hábiles, tal como pediste.


## Mejoras aplicadas
- Se refinó el tono visual para que se sienta más elegante y menos repetitivo con la palabra “premium”.
- Se añadió un bloque extra de valor para destacar la propuesta sin cambiar el alcance del paquete.
- Se incorporó el bono de la guía visual de WhatsApp Business como elemento visible de diferenciación.
- Se agregaron animaciones de entrada suaves al hacer scroll.
- Los enlaces de WhatsApp quedaron centralizados desde `assets/js/script.js` para editarlos una sola vez.
